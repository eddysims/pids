```ts

```

<Playground>
  <Heading as="h1">Heading 1</Heading>
  <Heading as="h2">Heading 2</Heading>
  <Heading as="h3">Heading 3</Heading>
</Playground>

## Props

<Props of={Heading} />

## Heading 1

A `Heading 1` should be used once per page. This should be used as the main title of the page and should appear as close to the top of the DOM as possible.

<Playground>
  <Heading as="h1">Heading 1</Heading>
</Playground>

## Themeing

| CSS Propery                                | Default                     | Description                                                |
| ------------------------------------------ | --------------------------- | ---------------------------------------------------------- |
| `--pids--font-family--display`             | System Font                 | The font family that will be used for all of the headings. |
| `--pids-components--heading--h1-font-size` | `--pids-font-size--huge`    | The font size that used for the prop `as="h1"`             |
| `--pids-components--heading--h2-font-size` | `--pids-font-size--largest` | The font size that used for the prop `as="h2"`             |
| `--pids-components--heading--h3-font-size` | `--pids-font-size--large`   | The font size that used for the prop `as="h3"`             |
